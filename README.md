# Gulpy

A simple, yet efficient Workspace using Gulp for web based projects.

## Features

- Create CSS & JavaScript bundles
- Compile SASS/SCSS into CSS
- Compile Pug into HTML
- Autoprefix CSS code
- Live reload
- Minify files.

## Dependencies

- Node.js
- yarn(or npm)

## Dev Envrionement

Install depedencies:

```lang-none
yarn
```

Run gulp & dev server:

```lang-none
yarn start dev
```
